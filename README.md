# Закрывающий тег

Одностраничный интерактивный веб-проект с карточной структурой, адаптивной вёрсткой, кастомной типографикой, CSS-анимациями и базовой JavaScript-логикой для обработки пользовательских действий.

## Демо

[Открыть проект на GitHub Pages](https://sonyales.github.io/zakrivayuschiy-teg-f/)

## Описание

Проект представляет собой стилизованную веб-страницу с набором карточек, визуальными эффектами для изображений, интерактивными кнопками Like и модальным окном. Интерфейс выполнен в пиксельной эстетике с использованием кастомных шрифтов, SVG-графики и CSS-переменных.

Основной акцент сделан на аккуратной HTML-структуре, гибкой системе стилей, адаптивности и плавных пользовательских взаимодействиях.

## Функциональность

- адаптивная вёрстка для разных размеров экрана;
- карточная структура контента;
- подключение кастомных шрифтов;
- использование CSS-переменных для управления цветами, отступами и типографикой;
- визуальные эффекты для изображений через CSS-фильтры;
- анимированная SVG-иконка Like;
- переключение состояния Like / Unlike через JavaScript;
- модальное окно с нативным элементом `dialog`;
- стилизованные состояния кнопок при наведении, фокусе и клике;
- поддержка доступного скрытия элементов через `visually-hidden`.

## Особенности реализации

### HTML

Разметка построена на семантических HTML-элементах. Контент организован в виде отдельных карточек, каждая из которых содержит заголовок, изображение, текстовый блок и интерактивные элементы управления.

Для модального окна используется нативный элемент `dialog`, что упрощает управление состоянием окна и повышает предсказуемость поведения интерфейса.

### CSS

Стили разделены по назначению:

- `globals.css` — базовые глобальные стили и сбросы;
- `variables.css` — CSS-переменные проекта;
- `style.css` — основные стили интерфейса;
- `animations.css` — анимации и состояния SVG-иконки Like.

В проекте используются CSS-переменные для централизованного управления цветами, размерами, отступами и параметрами типографики. Для адаптивности применяются медиавыражения и функция `clamp()`.

Изображения в карточках получают индивидуальные визуальные состояния с помощью CSS-фильтров: яркость, контрастность, насыщенность, размытие, инверсия и комбинированные эффекты.

### JavaScript

JavaScript используется для обработки интерактивности кнопок Like. Скрипт отслеживает клики по иконке и кнопке, переключает CSS-класс активного состояния и обновляет текст кнопки с `Like` на `Unlike`.

Логика вынесена в отдельный файл:

```txt
scripts/like.js
```

## Как запустить проект локально

Склонируйте репозиторий:

```bash
git clone https://github.com/sonyales/zakrivayuschiy-teg-f.git
```

Перейдите в папку проекта:

```bash
cd zakrivayuschiy-teg-f
```

Откройте файл `index.html` в браузере.

Дополнительная сборка или установка зависимостей не требуется.
